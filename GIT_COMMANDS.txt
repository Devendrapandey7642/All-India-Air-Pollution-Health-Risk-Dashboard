```powershell
git push -u origin main
```

## Step 5: GitHub Authentication
When prompted for password:
- Use your **Personal Access Token** from: https://github.com/settings/tokens
- NOT your GitHub password

### Create a Personal Access Token:
1. Go to: https://github.com/settings/tokens
2. Click "Generate new token (classic)"
3. Check boxes: `repo` (full control)
4. Click "Generate token"
5. **Copy the token** (it won't show again)
6. When git asks for password, **paste this token**
